## Dynamic Config

```yml
# http://traefik.digitalstudio.ltd/dashboard/#/
http:
  routers:
    dashboard-rtr:
      entryPoints:
        - web
        - websecure
      rule: Host(`traefik.digitalstudio.ltd`) && (PathPrefix(`/api`) || PathPrefix(`/dashboard`))
      service: api@internal
      middlewares:
        - middlewares-basic-auth
```
